## 📋 What is payload-nav-studio?

payload-nav-studio is a plugin for Payload CMS. It adds a sidebar navigation you can change to fit your needs. You can drag and drop menu items, choose from over 70 icons, and save your settings per user. It is ready for multiple languages, so you can use it in different regions.

This tool helps you simplify the admin panel. You don’t have to be a developer to use it. The sidebar can make your work faster by organizing links the way you want.
